UniGPT

Welcome to the official documentation of UniGPT!

FAQ

Q: What is UniGPT?

A: UniGPT is a self-hsoted solution for sharing OpenAI API access without directly sharing the API Key.

For game developers, we got Unity Editor Extension which can help with project analysis, asset management and pipeline.

For users, they can turn their phone/PC into a small server which is suitable for group studies and brainstorms.

Q: Is UniGPT another Discord-like or Siri-like app?

A: For standalone client, yes, it is for now. But for Unity Editor, it's more like a IDE.

Q: Can UniGPT help me visit OpenAI?

A: NO, IT WON'T WORK AS A VPN OR PROXY TO HELP YOU VISIT!

You need to figure out a way to let server access OpenAI.

Once server can access, the client can access too since all ChatCompletion requests are processed on server-side.

Q: How about the license and privacy?

A: We support BSD-3-Clause License and no data will be transferred to any third party except OpenAI.

This program is only suitable for research and internal projects development. For commercial usage, please contact both OpenAI and me.

Q: I want to help with the development/I got suggestion/I want to donate/I got issues with project, is that possible?

A: Yes, it is. Please contact me or join the Discord server.

Any help is appreciated!

User Manual

This section was divided into two part. Please find the corresponding part according to the version you use.

Standalone Client

Get Started

The only download source is on the github. Please visit https://github.com/Shepherd0619/UniGPT/releases.

For Windows, just download unigpt_setup.exe and follow the on screen instruction.

For Android, just download UniGPT_Android.apk and install it.

Right now, I only maintain Android and Windows version which I can be capable of. But you are free to compile for other platforms!

Host Role

In this case, you want to start a server but also join the server.

To start as a host, please type your nickname and press Login as Host.

Then, paste the OpenAI API Key you got from OpenAI Dashboard.

You can also change the UDP port (default is 7777) that server listen on.

For default, it will listen on all IP address.

Once you finished, click apply. It should direct ya to the ChatWindow.

If not, there will be a message box with detailed error info.

To stop the host/quit the app, press Esc (for PC) / Back (for Android).

Client Role

In this case, you want to connect to an exisiting server.

To start as a client, please type the server address and your nickname.

You can also upload an avatar if you want to.

If the connection is successful, you should be able to see the ChatWindow.

To disconnect from the server/quit the app, press Esc (for PC) / Back (for Android).

Interact with ChatGPT

Once you are in the ChatWindow, you can type your message in the inputfield down below.

Press Enter to create a new line. Press Ctrl+Enter to send the message.

You can also type /help to see available commands.

Chat Log/Chat History

For default, the server will save everyone's conversation.

You can continue your conversation by reconnecting to server with the nickname you used last time.

To delete the chat log, type /ClearChatLog.

To retrieve the chat log, type /GetChatLog. (This command is already executed when you successfully connect to a server.)

Copy/Share the message

To copy/share the message, please right click on the message (for PC) / touch the message and hold still for couple seconds (for Mobile). There should be a context menu shown next to your cursor/point you touched.

Please notice that share is only avaiable on mobile.

Troubleshooting

This section will only talk about the issue you may encounter in ChatWindow.

400 BAD REQUEST

Please make sure that:

1. You got valid OpenAI API Key.
2. You didn't reached the character limit.
3. Your OpenAI account is active and not banned.
4. Your network is safe and stable.

If you are pretty sure that there is nothing you can do, please contact me or create an issue on github.

Request Timeout

Please make sure that

1. You got valid network connection.
2. You got valid DNS server. (Recommend to use 8.8.8.8)
3. OpenAI is not under maintenance.

If you are pretty sure that there is nothing you can do, please contact OpenAI.

Disconnected immediately after connect

This is possibly caused by server or client.

If there is a fatal error on client-side, server will throw the client out.

if it is on server-side, server will shutdown and throw everyone out.

Please make sure that:

1. The version you use is matching the server's version.
2. The installation and the integrity are valid.

If you are pretty sure that there is nothing you can do, please contact the server's admin first.

If the admin can't help with it, please contact me or create an issue on github.
